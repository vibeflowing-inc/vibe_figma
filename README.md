# VibeFlow.ai - Figma to React Converter 

An implementation of VibeFlow.ai that converts Figma designs to React code using the official Figma API.

## About

VibeFlow.ai transforms your Figma designs into production-ready React code automatically. This tool leverages the official Figma API to accurately extract design information and generate clean, maintainable React components.

## Installation

To install dependencies:
```sh
bun install
```

## Usage

To run:
```sh
bun run dev
```

Open http://localhost:3000

## Features

- Official Figma API integration
- Automated Figma to React code conversion
- Clean, production-ready component generation
